O **UniMove** é um projeto de um aplicativo de transporte voluntário de universitário para universitário, trazendo preços mais acessiveis para pessoas que tenham necessidade de deslocamento para universidades e faculdades. A ideia é fazer com que pessoas que já vão de carro para faculdade, se volutanriem para levar outros alunos, e ainda ganhar uma pequeno renda extra.


## Objetivo do Projeto

O objetivo do projeto é aplicar conceitos de desenvolvimento de aplicativos e organização de projetos, criando um aplicativo de transporte para alunos de faculdades e universidades.

Durante o desenvolvimento foram utilizados conceitos como:

* Criação de um aplicativo
* Sistema de Login e cadastro
* Organização de corridas 
* Busca de voluntários que já vão de carro para a instituição de ensino
* Controle de versão com GitHub
* Organização de tarefas com Kanban

## Prazo e Custos

O projeto foi dividido em tarefas, as quais foram classificadas por prioridade, tamanho e "Pontos de Esforço". Para este último, as tarefas foram quantificadas em pontos para embasar uma hipótese de tempo e custos que estarão envolvidos no desenvolvimento do app. A seguir há uma tabela esclarecendo cada tarefa e suas classificações (também disponível na aba "Projects"):

Para transformar os pontos de esforço em tempo, é necessário definir uma "velocidade média" da equipe:
- Uma equipe pequena típica (1 dev mobile + 1 backend + 1 apoio/QA parcial) costuma ter uma velocidade média de 20 a 30 pontos por sprint.
- Vamos definir que cada sprint vale 2 semanas.
- Assim, vamos assumir uma velocidade mediana de 25 pontos por sprint.
- Logo, basta dividir o total de pontos pela velocidade mediana:
- 81 / 25 = 3,2 sprints.
- Vamos arredondar esse valor, assim adicionando margem, para 4 sprints.
- Como cada sprint vale 2 semanas, o prazo total estimado é de 8 a 10 semanas (adicionando mais uma margem).

Agora, para transformar o tempo em custo:
- Assumindo que a equipe será composta por 1 dev mobile + 1 backend + 1 apoio/QA parcial, estimaremos uma média de salário mensal individual de 8 mil reais.
- Transformando semanas em meses, teremos 2,5 meses (arredondaremos para 3 meses).
- Logo:
- (3 x 8000) x 3 = R$ 72.000,00 de custo.
