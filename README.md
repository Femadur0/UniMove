# UniMove

O **UniMove** é um projeto de um aplicativo de transporte voluntário de universitário para universitário, trazendo preços mais acessiveis para pessoas que tenham necessidade de deslocamento para universidades e faculdades. A ideia é fazer com que pessoas que já vão de carro para faculdad, se volutanriem para levar outros alunos, e ainda ganhar uma pequeno renda extra.

O sistema permite que usuários se cadastrem, façam login, visualizem produtos e adicionem itens ao carrinho.

## Objetivo do Projeto

O objetivo do projeto é aplicar conceitos de desenvolvimento web e organização de projetos, criando uma loja virtual simples para venda de doces.

Durante o desenvolvimento foram utilizados conceitos como:

* Criação de páginas web
* Sistema de login e cadastro
* Organização de produtos
* Carrinho de compras
* Controle de versão com GitHub
* Organização de tarefas com Kanban

## 🛠️ Tecnologias Utilizadas

* HTML5

* PHP
* MySQL
* Git
* GitHub

## ⚙️ Funcionalidades do Sistema

* Cadastro de usuários
* Login de usuários
* Visualização de produtos
* Organização de produtos por categoria
* Adição de produtos ao carrinho

## 📋 Metodologia

O projeto utiliza princípios de **metodologia ágil**, organizando o desenvolvimento através de:

* Histórias de usuário
